- JavaScript
- GraphQL API
- Apollo Server
- Stripe API
- React Redux
- IndexedDB

## Installation

To run this project, install it locally using npm:

```
npm install
```

## Usage

the application will be invoked by using the following command:

```
npm run develop
```
